
1. enter virtual environment
```
source .venv/bin/activate
```

2. run as development server
```bash
mkdocs serve
```

3. deploy to github pages
```bash
mkdocs deploy
```

---

3. add files to git index
```bash
git add <file_name>
```

4. commit file  with message
```bash
git commit -m "<message>"
```

5. upload to github
```bash
git push origin <branch_name>
```
